# Bot Manajemen Keuangan & Dashboard

Aplikasi ini menggabungkan Bot Telegram dan Dashboard Web untuk mencatat transaksi keuangan harian secara real-time.

## Fitur
- Input transaksi via Telegram.
- Perhitungan otomatis (Kurs & Fee).
- Dashboard Web untuk melihat laporan & mengubah pengaturan.

## Cara Install
1. Clone repo ini.
2. Install dependencies: `pip install -r requirements.txt`
3. Jalankan Website: `python app.py`
4. Buka Web (http://127.0.0.1:5000), masuk ke Settings, isi Token Bot.
5. Jalankan Bot: `python bot.py`
